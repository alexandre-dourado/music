Como usar arquivos locais:
Coloque os arquivos nas pastas dentro de public/:
• audios/nome.mp3 → coluna audio_id
• letras/nome.txt → coluna letra_id
• capas/nome.jpg → coluna capa_id
Os nomes devem ser iguais (sem extensão).

📂 Atualizar planilha com caminhos locais

Corrige as colunas audio_id, letra_id e capa_id das músicas existentes para usar os caminhos locais baseado no título.

📂 Escanear pasta /audios do Drive

Importa arquivos do Google Drive (modo antigo).

🔄 Limpar Cache

Força releitura da planilha.

IDs salvos como constantes no Code.gs. Para alterar, edite diretamente no Apps Script.
